# Java 库类

这个指导将包含 java.lang 包，它提供了对 Java 编程语言的设计非常重要的类。最重要的类是 Object，它是类层次的根。还有 Class,它是代表运行时类的实例。  

这是 java.lang 包的类列表。这些类对一个 Java 编程者来说是非常重要的。点击一个类的链接来知道更多有关类的细节。更一步钻研的话，你可以参考标准 Java 文档。
